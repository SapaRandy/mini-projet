Lancement du mini-projet Santa Claus

Ce projet à pour but de réaliser la gestion de la distribution de cadeaux par le père Noel avec l’utilisation des design patterns Facade, Strategy, Decorator, Factory, Observer et Command.

Prérequis
Python 3 installé (3.8 ou plus récent recommandé).

Les fichiers suivants présents dans le même dossier :

command.py
decorateur.py
facade.py
factory.py
observer.py
strategy.py
main.py contenant le code de lancement.


Installation
Clone le dépôt ou copie les fichiers dans un dossier, par exemple :

bash
git clone https://github.com/SapaRandy/mini-projet.git

Vérifie que Python est accessible :

bash
python --version
ou selon ta configuration :

bash
python3 --version

Exécution
Dans le dossier du projet, exécute :

bash
python main.py

ou :

bash
python3 main.py

Tu devrais obtenir une sortie similaire dans la console :

Elf Stark notified: Toy wrapped with ribbon with message 'Ho Ho Ho!'
Toy wrapped with ribbon with message 'Ho Ho Ho!'
Delivered by reindeer
--------------------------------------------------
Elf Stark notified: Book wrapped with ribbon with message 'Ho Ho Ho!'
Book wrapped with ribbon with message 'Ho Ho Ho!'
Delivered by drone
